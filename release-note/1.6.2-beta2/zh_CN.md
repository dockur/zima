## [1.6.2-beta2]

### 修复
- 修复部分系统目录被错误识别为无效路径的问题

### 优化
- 优化网盘在特定场景下的初始化配置流程，修复因初始化过慢导致 Files 加载异常的问题

## [1.6.2-beta1]

### 安全
- 紧急修复账户管理模块中的高风险安全问题，进一步提升管理员操作安全性
- 紧急修复文件服务中的高风险安全问题，提升文件访问与操作安全性

### 新增
- 新增 Files 应用深色模式
- 新增阿里云盘接入支持
- 更新 Linux 内核版本至 6.18.9
- 新增对 NVIDIA 470 / 580 / Open Kernel Module、Intel Xe / Battlemage、Realtek RTW88 / RTL8127、Intel / MediaTek / AIC8800 无线设备及 Cirrus Logic 音频设备的驱动支持，并新增 NVIDIA 驱动自动检测与适配机制，可根据硬件自动匹配合适的驱动版本。

### 修复
- 修复 OneDrive 密钥过期导致的连接问题，恢复基础接入能力，并优化密钥更新方式
- 修复 Collabora Server 应用阻断备份的问题，确保备份任务可以正常执行
- 修复启动时多个 USB 存储可能被分配到同一挂载点的问题
- 修复 USB 挂载点冲突时生成旧式 `-设备名` 后缀的问题
- 修复单分区磁盘创建存储时未使用用户指定名称作为挂载点的问题
- 修复相同序列号硬盘组成 RAID 后重启降级的问题
- 修复 RAID 启动检查中运行态设备名格式不一致导致的误判

### 优化
- 优化导入、迁移、加密、复制/剪切等高频文件操作流程
- 优化云盘上传状态展示，让用户更清楚地了解当前进度、完成状态和失败原因
