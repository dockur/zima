## [1.6.2]

### Security
- Διορθώθηκε επειγόντως ένα ζήτημα ασφαλείας υψηλού κινδύνου στη μονάδα διαχείρισης λογαριασμών, βελτιώνοντας περαιτέρω την ασφάλεια των ενεργειών διαχειριστή
- Διορθώθηκε επειγόντως ένα ζήτημα ασφαλείας υψηλού κινδύνου στην υπηρεσία αρχείων, βελτιώνοντας την ασφάλεια της πρόσβασης και των λειτουργιών αρχείων
- Διορθώθηκε ένα ζήτημα ασφαλείας υψηλού κινδύνου στην υπηρεσία μηνυμάτων, βελτιώνοντας την ασφάλεια της επικοινωνίας του συστήματος
- Το SSH πλέον απενεργοποιεί από προεπιλογή τη σύνδεση root από απόσταση, μειώνοντας τον κίνδυνο έκθεσης λογαριασμών υψηλών δικαιωμάτων
- Το SSH έχει προσθέσει αρκετές πολιτικές ασφαλείας για την αποτροπή επιθέσεων brute-force
- Η υπηρεσία NFS είναι πλέον απενεργοποιημένη από προεπιλογή, μειώνοντας τον κίνδυνο μη εξουσιοδοτημένης πρόσβασης

### New
- Προστέθηκε σκοτεινή λειτουργία για την εφαρμογή Files
- Προστέθηκε υποστήριξη ενσωμάτωσης με το Alibaba Cloud Drive
- Ο πυρήνας Linux ενημερώθηκε στην έκδοση 6.18.9
- Προστέθηκε υποστήριξη οδηγών για NVIDIA 470 / 580 / Open Kernel Module, Intel Xe / Battlemage, Realtek RTW88 / RTL8127, ασύρματες συσκευές Intel / MediaTek / AIC8800 και συσκευές ήχου Cirrus Logic, καθώς και μηχανισμός αυτόματης ανίχνευσης και προσαρμογής οδηγού NVIDIA που αντιστοιχίζει αυτόματα την κατάλληλη έκδοση οδηγού με βάση το υλικό

### Fixed
- Διορθώθηκε το πρόβλημα σύνδεσης που προκαλούνταν από τη λήξη του κλειδιού του OneDrive, αποκαταστάθηκε η βασική διαθεσιμότητα της ενσωμάτωσης και βελτιστοποιήθηκε ο μηχανισμός ενημέρωσης του κλειδιού
- Διορθώθηκε ένα πρόβλημα όπου η εφαρμογή Collabora Server εμπόδιζε τα αντίγραφα ασφαλείας, διασφαλίζοντας ότι οι εργασίες αντιγράφων ασφαλείας μπορούν να εκτελούνται σωστά
- Διορθώθηκε ένα πρόβλημα όπου σε πολλές συσκευές αποθήκευσης USB μπορούσε να εκχωρηθεί το ίδιο σημείο προσάρτησης κατά την εκκίνηση
- Διορθώθηκε ένα πρόβλημα όπου δημιουργούνταν παλαιά επιθήματα `-device-name` όταν υπήρχαν συγκρούσεις σημείων προσάρτησης USB
- Διορθώθηκε η υποβάθμιση RAID μετά από επανεκκίνηση όταν μια συστοιχία RAID δημιουργούνταν από δίσκους με τον ίδιο σειριακό αριθμό

### Optimized
- Βελτιστοποιήθηκαν οι συχνές ροές εργασιών αρχείων, όπως εισαγωγή, μετεγκατάσταση, κρυπτογράφηση, αντιγραφή και αποκοπή
- Βελτιστοποιήθηκε η εμφάνιση της κατάστασης μεταφόρτωσης στο cloud drive, ώστε οι χρήστες να κατανοούν πιο καθαρά την τρέχουσα πρόοδο, την κατάσταση ολοκλήρωσης και τους λόγους αποτυχίας
