## [1.6.2]

### Security
- Срочно устранена уязвимость высокого риска в модуле управления учетными записями, что дополнительно повысило безопасность действий администратора
- Срочно устранена уязвимость высокого риска в файловом сервисе, что повысило безопасность доступа к файлам и операций с ними
- Устранена уязвимость высокого риска в службе сообщений, что повысило безопасность системных коммуникаций
- SSH теперь по умолчанию отключает удаленный вход для пользователя root, снижая риск раскрытия высокопривилегированных учетных записей
- В SSH добавлено несколько политик безопасности для предотвращения атак brute-force
- Служба NFS теперь по умолчанию отключена, что снижает риск несанкционированного доступа

### New
- Добавлен темный режим для приложения Files
- Добавлена поддержка интеграции с Alibaba Cloud Drive
- Ядро Linux обновлено до версии 6.18.9
- Добавлена поддержка драйверов для NVIDIA 470 / 580 / Open Kernel Module, Intel Xe / Battlemage, Realtek RTW88 / RTL8127, беспроводных устройств Intel / MediaTek / AIC8800 и аудиоустройств Cirrus Logic, а также добавлен механизм автоматического определения и адаптации драйверов NVIDIA, который автоматически подбирает подходящую версию драйвера в зависимости от оборудования

### Fixed
- Исправлена проблема подключения, вызванная истечением срока действия ключа OneDrive, восстановлена базовая доступность интеграции и оптимизирован механизм обновления ключа
- Исправлена проблема, из-за которой приложение Collabora Server блокировало резервное копирование, что гарантирует корректную работу задач резервного копирования
- Исправлена проблема, из-за которой при запуске нескольким USB-накопителям могла назначаться одна и та же точка монтирования
- Исправлена проблема, из-за которой при конфликте точек монтирования USB создавались устаревшие суффиксы `-device-name`
- Исправлена деградация RAID после перезагрузки, когда RAID-массив создавался из дисков с одинаковым серийным номером

### Optimized
- Оптимизированы высокочастотные сценарии работы с файлами, такие как импорт, миграция, шифрование, копирование и вырезание
- Оптимизировано отображение статуса загрузки в облачное хранилище, чтобы пользователям было понятнее текущее состояние прогресса, завершения и причины ошибок
