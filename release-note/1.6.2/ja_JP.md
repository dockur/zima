## [1.6.2]

### Security
- アカウント管理モジュールの高リスクなセキュリティ問題を緊急修正し、管理者操作の安全性をさらに向上させました
- ファイルサービスの高リスクなセキュリティ問題を緊急修正し、ファイルアクセスと操作の安全性を向上させました
- メッセージサービスの高リスクなセキュリティ問題を修正し、システム通信の安全性を向上させました
- SSH は root ユーザーのリモートログインをデフォルトで無効化し、高権限アカウントの露出リスクを低減します
- SSH に総当たり攻撃を防ぐための複数のセキュリティポリシーを追加しました
- NFS サービスはデフォルトで無効になり、不正アクセスのリスクを低減します

### New
- Files アプリにダークモードを追加しました
- Alibaba Cloud Drive の統合サポートを追加しました
- Linux カーネルを 6.18.9 に更新しました
- NVIDIA 470 / 580 / Open Kernel Module、Intel Xe / Battlemage、Realtek RTW88 / RTL8127、Intel / MediaTek / AIC8800 無線デバイス、Cirrus Logic オーディオデバイス向けのドライバーサポートを追加し、ハードウェアに応じて適切なドライバー版を自動的に割り当てる NVIDIA ドライバー自動検出・適応メカニズムも追加しました

### Fixed
- OneDrive キーの期限切れによる接続問題を修正し、基本的な統合機能を復旧するとともに、キー更新の仕組みを最適化しました
- Collabora Server アプリがバックアップをブロックする問題を修正し、バックアップタスクが正常に実行できるようにしました
- 起動時に複数の USB ストレージデバイスへ同じマウントポイントが割り当てられる問題を修正しました
- USB マウントポイントが競合した際に従来形式の `-device-name` 接尾辞が生成される問題を修正しました
- 同じシリアル番号のディスクから RAID アレイを作成した場合に、再起動後 RAID が劣化状態になる問題を修正しました

### Optimized
- インポート、移行、暗号化、コピー、切り取りなど高頻度なファイル操作のワークフローを最適化しました
- クラウドドライブのアップロード状態表示を最適化し、現在の進行状況、完了状態、失敗理由をより分かりやすくしました
