## [1.6.2]

### Security
- Ceartaíodh go práinneach saincheist slándála ardriosca sa mhodúl bainistíochta cuntas, rud a fheabhsaíonn tuilleadh sábháilteacht oibríochtaí riarthóra
- Ceartaíodh go práinneach saincheist slándála ardriosca sa tseirbhís comhad, rud a fheabhsaíonn sábháilteacht rochtana agus oibríochtaí comhad
- Ceartaíodh saincheist slándála ardriosca sa tseirbhís teachtaireachtaí, rud a fheabhsaíonn slándáil chumarsáide an chórais
- Díchumasaíonn SSH anois logáil isteach cianda don úsáideoir root de réir réamhshocraithe, rud a laghdaíonn riosca nochtaithe cuntas ardphribhléide
- Chuir SSH roinnt polasaithe slándála leis chun ionsaithe brute-force a chosc
- Tá an tseirbhís NFS díchumasaithe de réir réamhshocraithe anois, rud a laghdaíonn an baol rochtana neamhúdaraithe

### New
- Cuireadh mód dorcha leis don aip Files
- Cuireadh tacaíocht chomhtháthaithe Alibaba Cloud Drive leis
- Nuashonraíodh eithne Linux go leagan 6.18.9
- Cuireadh tacaíocht tiománaithe leis do NVIDIA 470 / 580 / Open Kernel Module, Intel Xe / Battlemage, Realtek RTW88 / RTL8127, gléasanna gan sreang Intel / MediaTek / AIC8800, agus gléasanna fuaime Cirrus Logic, agus cuireadh meicníocht bhrath agus oiriúnaithe uathoibríoch do thiománaithe NVIDIA leis a mheaitseálann an leagan tiománaí cuí go huathoibríoch bunaithe ar an gcrua-earra

### Fixed
- Ceartaíodh an fhadhb cheangail a bhí mar thoradh ar eochair OneDrive a bheith imithe in éag, cuireadh ar ais bun-infhaighteacht an chomhtháthaithe, agus rinneadh an meicníocht nuashonraithe eochrach a bharrfheabhsú
- Ceartaíodh saincheist ina raibh an aip Collabora Server ag cur bac ar chúltacaí, rud a chinntíonn gur féidir tascanna cúltaca a rith i gceart
- Ceartaíodh saincheist ina bhféadfaí an pointe gléasta céanna a shannadh do roinnt gléasanna stórála USB ag am tosaithe
- Ceartaíodh saincheist ina ngintear iarmhíreanna oidhreachta `-device-name` nuair a bhí coimhlintí ag pointí gléasta USB
- Ceartaíodh díghrádú RAID tar éis atosaithe nuair a cruthaíodh eagar RAID as dioscaí leis an uimhir sraithe chéanna

### Optimized
- Rinneadh sreafaí oibre ardoibríochtaí comhad a bharrfheabhsú, lena n-áirítear iompórtáil, imirce, criptiú, cóipeáil, agus gearradh
- Rinneadh taispeáint stádais uaslódála an tiomántáin néil a bharrfheabhsú, rud a fhágann go bhfuil sé níos soiléire d'úsáideoirí an dul chun cinn reatha, an stádas críochnaithe agus cúiseanna teipe a thuiscint
