## [1.6.2]

### Security
- 계정 관리 모듈의 고위험 보안 문제를 긴급 수정하여 관리자 작업의 안전성을 한층 더 높였습니다
- 파일 서비스의 고위험 보안 문제를 긴급 수정하여 파일 접근 및 작업의 안전성을 높였습니다
- 메시징 서비스의 고위험 보안 문제를 수정하여 시스템 통신 보안성을 높였습니다
- SSH는 이제 기본적으로 root 사용자의 원격 로그인을 비활성화하여 고권한 계정 노출 위험을 줄입니다
- SSH에 무차별 대입 공격을 방지하기 위한 여러 보안 정책을 추가했습니다
- NFS 서비스는 이제 기본적으로 비활성화되어 무단 접근 위험을 줄입니다

### New
- Files 앱에 다크 모드를 추가했습니다
- Alibaba Cloud Drive 연동 지원을 추가했습니다
- Linux 커널을 6.18.9로 업데이트했습니다
- NVIDIA 470 / 580 / Open Kernel Module, Intel Xe / Battlemage, Realtek RTW88 / RTL8127, Intel / MediaTek / AIC8800 무선 장치, Cirrus Logic 오디오 장치에 대한 드라이버 지원을 추가했으며, 하드웨어에 따라 적절한 드라이버 버전을 자동으로 매칭하는 NVIDIA 드라이버 자동 감지 및 적응 메커니즘도 추가했습니다

### Fixed
- OneDrive 키 만료로 인한 연결 문제를 수정하고 기본 연동 기능을 복구했으며 키 업데이트 메커니즘을 최적화했습니다
- Collabora Server 앱이 백업을 차단하던 문제를 수정하여 백업 작업이 정상적으로 실행되도록 했습니다
- 시작 시 여러 USB 저장 장치에 동일한 마운트 지점이 할당될 수 있던 문제를 수정했습니다
- USB 마운트 지점 충돌 시 기존 형식의 `-device-name` 접미사가 생성되던 문제를 수정했습니다
- 동일한 일련번호를 가진 디스크로 RAID 배열을 생성했을 때 재부팅 후 RAID가 성능 저하 상태가 되던 문제를 수정했습니다

### Optimized
- 가져오기, 마이그레이션, 암호화, 복사, 잘라내기 등 고빈도 파일 작업 흐름을 최적화했습니다
- 클라우드 드라이브 업로드 상태 표시를 최적화하여 현재 진행 상황, 완료 상태, 실패 원인을 사용자가 더 명확하게 이해할 수 있도록 했습니다
