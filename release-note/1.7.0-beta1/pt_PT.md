## [1.7.0-beta1]

### Added
- A interface da AppStore foi redesenhada para proporcionar uma experiência de navegação de aplicações mais moderna e mais clara
- Foi adicionada uma página de gestão de aplicações para gerir centralmente as aplicações instaladas, o estado das atualizações e as ações mais comuns
- Foi adicionada uma página de descoberta de aplicações com recomendações melhoradas, navegação por categorias e apresentação de conteúdos
- Foi adicionado suporte nativo a YAML para melhorar a edição e a análise de configurações

### Improved
- A experiência de apresentação das listas de aplicações, páginas de categorias e páginas de detalhes das aplicações foi melhorada
- A apresentação dos registos dos contentores foi melhorada para oferecer melhor legibilidade e uma resolução de problemas mais eficiente
- As operações do terminal do contentor foram melhoradas com um estado de execução mais claro e melhor feedback interativo
- A lógica de validação das licenças comerciais foi melhorada para reforçar a compatibilidade e a precisão do reconhecimento de licenças
- A estratégia de limpeza da cache das páginas foi melhorada para reforçar a experiência geral de navegação
