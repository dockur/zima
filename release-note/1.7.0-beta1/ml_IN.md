## [1.7.0-beta1]

### Added
- കൂടുതല്‍ ആധുനികവും വ്യക്തവുമായ ആപ്പ് ബ്രൗസിംഗ് അനുഭവത്തിനായി AppStore UI പുനര്‍രൂപകല്‍പന ചെയ്തു
- ഇന്‍സ്റ്റാള്‍ ചെയ്ത ആപ്പുകള്‍, അപ്ഡേറ്റ് നില, സാധാരണ പ്രവര്‍ത്തനങ്ങള്‍ എന്നിവ കേന്ദ്രമായി കൈകാര്യം ചെയ്യാന്‍ ഒരു ആപ്പ് മാനേജ്മെന്റ് പേജ് ചേര്‍ത്തു
- മെച്ചപ്പെടുത്തിയ ആപ്പ് ശുപാര്‍ശകള്‍, വിഭാഗ ബ്രൗസിംഗ്, ഉള്ളടക്ക അവതരണം എന്നിവയോടുകൂടിയ ഒരു ആപ്പ് കണ്ടെത്തല്‍ പേജ് ചേര്‍ത്തു
- കോണ്‍ഫിഗറേഷന്‍ എഡിറ്റിംഗും പാര്‍സിംഗും മെച്ചപ്പെടുത്തുന്നതിനായി നേറ്റീവ് YAML പിന്തുണ ചേര്‍ത്തു

### Improved
- ആപ്പ് ലിസ്റ്റുകള്‍, വിഭാഗ പേജുകള്‍, ആപ്പ് വിശദാംശ പേജുകള്‍ എന്നിവയുടെ പ്രദര്‍ശന അനുഭവം മെച്ചപ്പെടുത്തി
- മികച്ച വായനാസൗകര്യത്തിനും കൂടുതല്‍ കാര്യക്ഷമമായ പ്രശ്നപരിഹാരത്തിനുമായി കണ്ടെയ്‌നര്‍ ലോഗ് പ്രദര്‍ശനം മെച്ചപ്പെടുത്തി
- കൂടുതല്‍ വ്യക്തമായ എക്സിക്യൂഷന്‍ നിലയും ഇടപെടല്‍ പ്രതികരണവും നല്‍കി കണ്ടെയ്‌നര്‍ ടെര്‍മിനല്‍ പ്രവര്‍ത്തനങ്ങള്‍ മെച്ചപ്പെടുത്തി
- ലൈസന്‍സ് തിരിച്ചറിവിന്റെ പൊരുത്തക്ഷമതയും കൃത്യതയും വര്‍ധിപ്പിക്കാന്‍ കൊമേഴ്ഷ്യല്‍ ലൈസന്‍സ് സാധൂകരണ ലജിക്ക് മെച്ചപ്പെടുത്തി
- മൊത്തത്തിലുള്ള ബ്രൗസിംഗ് അനുഭവം മെച്ചപ്പെടുത്തുന്നതിനായി പേജ് കാഷ് ശുദ്ധീകരണ തന്ത്രം മെച്ചപ്പെടുത്തി
