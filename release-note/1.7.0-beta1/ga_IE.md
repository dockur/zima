## [1.7.0-beta1]

### Added
- Rinneadh athdhearadh ar chomhéadan AppStore chun eispéireas níos nua-aimseartha agus níos soiléire a thabhairt agus aipeanna á mbrabhsáil
- Cuireadh leathanach bainistíochta aipeanna leis chun aipeanna suiteáilte, stádas nuashonruithe agus gníomhartha coitianta a bhainistiú go lárnach
- Cuireadh leathanach fionnachtana aipeanna leis le moltaí feabhsaithe, brabhsáil catagóirí agus cur i láthair ábhair níos fearr
- Cuireadh tacaíocht dhúchasach YAML leis chun eagarthóireacht agus parsáil cumraíochtaí a fheabhsú

### Improved
- Feabhsaíodh an taithí taispeána do liostaí aipeanna, leathanaigh chatagóire agus leathanaigh shonraí aipeanna
- Feabhsaíodh taispeáint logaí coimeádán le haghaidh inléiteachta níos fearr agus fabhtcheartaithe níos éifeachtaí
- Feabhsaíodh oibríochtaí teirminéil coimeádán le stádas forghníomhaithe níos soiléire agus aiseolas idirghníomhach níos fearr
- Feabhsaíodh loighic bailíochtaithe ceadúnas tráchtála chun comhoiriúnacht agus cruinneas aitheantais ceadúnas a neartú
- Feabhsaíodh straitéis glanta taisce leathanaigh chun an t-eispéireas brabhsála foriomlán a fheabhsú
