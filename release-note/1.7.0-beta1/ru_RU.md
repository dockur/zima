## [1.7.0-beta1]

### Added
- Интерфейс AppStore был переработан для более современного и понятного просмотра приложений
- Добавлена страница управления приложениями для централизованного управления установленными приложениями, статусом обновлений и типовыми действиями
- Добавлена страница поиска приложений с улучшенными рекомендациями, просмотром по категориям и представлением контента
- Добавлена нативная поддержка YAML для улучшения редактирования и разбора конфигураций

### Improved
- Улучшено отображение списков приложений, страниц категорий и страниц сведений о приложениях
- Улучшено отображение журналов контейнеров для повышения читаемости и более эффективного поиска неисправностей
- Улучшены операции терминала контейнеров за счет более понятного отображения статуса выполнения и интерактивной обратной связи
- Улучшена логика проверки коммерческих лицензий для повышения совместимости и точности распознавания лицензий
- Улучшена стратегия очистки кэша страниц для улучшения общего опыта просмотра
