## [1.7.0-beta1]

### Added
- AppStore の UI を刷新し、よりモダンで分かりやすいアプリ閲覧体験を実現しました
- インストール済みアプリ、更新状況、よく使う操作を一元管理できるアプリ管理ページを追加しました
- アプリのおすすめ、カテゴリ閲覧、コンテンツ表示を改善したアプリ発見ページを追加しました
- 設定の編集と解析を改善するため、ネイティブな YAML サポートを追加しました

### Improved
- アプリ一覧、カテゴリページ、アプリ詳細ページの表示体験を改善しました
- 読みやすさとトラブルシューティング効率を高めるため、コンテナログの表示を改善しました
- 実行状態の表示と操作フィードバックをより明確にし、コンテナターミナル操作を改善しました
- ライセンス認識の互換性と精度を高めるため、商用ライセンス検証ロジックを改善しました
- ページキャッシュのクリーンアップ戦略を改善し、全体的なブラウジング体験を向上させました
