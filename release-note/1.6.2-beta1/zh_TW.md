## [1.6.2-beta1]

### 安全
- 緊急修復帳戶管理模組中的高風險安全問題，進一步提升管理員操作安全性
- 緊急修復檔案服務中的高風險安全問題，提升檔案存取與操作安全性

### 新增
- 新增 Files 應用程式深色模式
- 新增阿里雲盤接入支援
- 更新 Linux 核心版本至 6.18.9
- 新增對 NVIDIA 470 / 580 / Open Kernel Module、Intel Xe / Battlemage、Realtek RTW88 / RTL8127、Intel / MediaTek / AIC8800 無線裝置及 Cirrus Logic 音訊裝置的驅動支援，並新增 NVIDIA 驅動自動偵測與適配機制，可根據硬體自動匹配合適的驅動版本。

### 修復
- 修復 OneDrive 金鑰過期導致的連線問題，恢復基礎接入能力，並優化金鑰更新方式
- 修復 Collabora Server 應用程式阻斷備份的問題，確保備份任務可以正常執行
- 修復啟動時多個 USB 儲存裝置可能被分配到同一掛載點的問題
- 修復 USB 掛載點衝突時產生舊式 `-裝置名稱` 後綴的問題
- 修復單分割區磁碟建立儲存時未使用使用者指定名稱作為掛載點的問題
- 修復相同序號硬碟組成 RAID 後重啟降級的問題
- 修復 RAID 啟動檢查中執行階段裝置名稱格式不一致導致的誤判

### 優化
- 優化匯入、遷移、加密、複製/剪下等高頻檔案操作流程
- 優化雲端硬碟上傳狀態顯示，讓使用者更清楚了解目前進度、完成狀態和失敗原因
