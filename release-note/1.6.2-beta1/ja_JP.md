## [1.6.2-beta1]

### Security
- アカウント管理モジュールの高リスクのセキュリティ問題が緊急修正され、管理者の操作の安全性がさらに向上しました。
- ファイル サービスのリスクの高いセキュリティ問題を緊急に修正し、ファイル アクセスと操作の安全性を向上させました。

### New
- ファイルアプリにダークモードを追加
- Alibaba Cloud Drive統合サポートを追加
- Linux カーネルを 6.18.9 に更新しました
- NVIDIA 470 / 580 / Open Kernel Module、Intel Xe / Battlemage、Realtek RTW88 / RTL8127、Intel / MediaTek / AIC8800 ワイヤレス デバイス、および Cirrus Logic オーディオ デバイスのドライバー サポートを追加し、ハードウェアに基づいて適切なドライバー バージョンを自動的に照合する NVIDIA ドライバーの自動検出および適応メカニズムを追加しました。

### Fixed
- 接続の問題を引き起こす OneDrive キーの有効期限を修正し、基本的な統合の可用性を復元し、キー更新メカニズムを最適化しました。
- Collabora サーバー アプリがバックアップをブロックする問題を修正し、バックアップ タスクが適切に実行できるようにしました。
- 起動時に複数の USB ストレージ デバイスに同じマウント ポイントが割り当てられる問題を修正しました。
- USB マウント ポイントが競合したときに従来の `-device-name` サフィックスが生成される問題を修正しました
- 単一パーティションのディスクから作成されたストレージがユーザー指定の名前をマウント ポイントとして使用しない問題を修正しました。
- 同じシリアル番号を持つディスクから RAID アレイを作成した場合の、再起動後の RAID の劣化を修正しました。
- 一貫性のないランタイムデバイス名の形式によって引き起こされる RAID 起動チェックでの誤検知を修正しました。

### Optimized
- インポート、移行、暗号化、コピー、切り取りなど、高頻度のファイル操作ワークフローを最適化
- クラウド ドライブのアップロード ステータス表示が最適化され、ユーザーが現在の進行状況、完了ステータス、失敗の理由をより明確に理解できるようになりました。
