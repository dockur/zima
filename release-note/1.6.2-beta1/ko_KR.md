## [1.6.2-beta1]

### Security
- 계정 관리 모듈의 고위험 보안 문제를 긴급 수정하여 관리자 운영의 안전성을 더욱 향상시켰습니다.
- 파일 서비스의 고위험 보안 문제를 긴급 수정하여 파일 액세스 및 작업의 안전성을 향상했습니다.

### New
- 파일 앱에 어두운 모드가 추가되었습니다.
- Alibaba Cloud Drive 통합 지원 추가
- Linux 커널을 6.18.9로 업데이트했습니다.
- NVIDIA 470/580/개방형 커널 모듈, Intel Xe/Battlemage, Realtek RTW88/RTL8127, Intel/MediaTek/AIC8800 무선 장치 및 Cirrus Logic 오디오 장치에 대한 드라이버 지원을 추가하고 하드웨어에 따라 적절한 드라이버 버전을 자동으로 일치시키는 NVIDIA 드라이버 자동 감지 및 적응 메커니즘을 추가했습니다.

### Fixed
- 연결 문제를 일으키는 OneDrive 키 만료 문제를 수정하고, 기본 통합 가용성을 복원하고, 키 업데이트 메커니즘을 최적화했습니다.
- Collabora Server 앱이 백업을 차단하여 백업 작업이 제대로 실행될 수 있도록 하는 문제를 수정했습니다.
- 시작 시 여러 USB 저장 장치에 동일한 마운트 지점이 할당될 수 있는 문제가 수정되었습니다.
- USB 마운트 지점이 충돌할 때 레거시 '-device-name' 접미사가 생성되는 문제를 해결했습니다.
- 단일 파티션 디스크에서 생성된 스토리지가 사용자 지정 이름을 마운트 지점으로 사용하지 않는 문제를 해결했습니다.
- 동일한 일련 번호를 가진 디스크에서 RAID 어레이가 생성된 경우 재부팅 후 RAID 저하 문제를 수정했습니다.
- 일관되지 않은 런타임 장치 이름 형식으로 인해 발생하는 RAID 시작 검사의 잘못된 긍정을 수정했습니다.

### Optimized
- 가져오기, 마이그레이션, 암호화, 복사, 잘라내기 등 최적화된 고주파 파일 작업 워크플로우
- 최적화된 클라우드 드라이브 업로드 상태 표시를 통해 사용자가 현재 진행 상황, 완료 상태 및 실패 이유를 보다 명확하게 이해할 수 있습니다.
