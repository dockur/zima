## [1.7.0-beta2]

### Fixed
- 멤버 사용자에게 앱 목록이 표시되지 않던 문제를 수정했습니다
- 특정 네트워크 환경에서 앱 목록이 로드되지 않던 문제를 수정했습니다
- 환경 변수 내 따옴표 처리 오류로 인해 앱이 업데이트되거나 설치되지 않던 문제를 수정했습니다
- 일부 로캘에서 메모리 값 형식이 잘못되어 앱 설치에 실패하던 문제를 수정했습니다
- 작은 화면 장치에서 일부 창이 완전히 보이지 않던 문제를 수정했습니다
- 백업 페이지에서 파일 수 표시가 겹치던 문제를 수정했습니다
- 대용량 파일 백업 중 지연이 발생하던 문제를 수정했습니다

## [1.7.0-beta1]

### Added
- 더 현대적이고 명확한 앱 탐색 경험을 위해 AppStore UI를 새롭게 개편했습니다
- 설치된 앱, 업데이트 상태, 자주 사용하는 작업을 중앙에서 관리할 수 있는 앱 관리 페이지를 추가했습니다
- 앱 추천, 카테고리 탐색, 콘텐츠 표시를 개선한 앱 탐색 페이지를 추가했습니다
- 구성 편집과 파싱 경험을 향상시키기 위해 기본 YAML 지원을 추가했습니다

### Improved
- 앱 목록, 카테고리 페이지, 앱 상세 페이지의 표시 경험을 개선했습니다
- 더 나은 가독성과 효율적인 문제 해결을 위해 컨테이너 로그 표시를 개선했습니다
- 더 명확한 실행 상태와 상호작용 피드백으로 컨테이너 터미널 작업을 개선했습니다
- 라이선스 인식의 호환성과 정확성을 높이기 위해 상용 라이선스 검증 로직을 개선했습니다
- 전반적인 탐색 경험을 향상시키기 위해 페이지 캐시 정리 전략을 개선했습니다
