## [1.7.0-beta2]

### 修復
- 修復成員使用者的應用程式列表未顯示的問題
- 修復在特定網路環境下，應用程式列表可能無法載入的問題
- 修復環境變數中引號處理異常，導致應用程式無法修改或安裝的問題
- 修復部分語言環境下記憶體數值格式異常，導致應用程式無法安裝的問題
- 修復小螢幕裝置上部分視窗顯示不完整的問題
- 修復備份頁面中文件數量顯示重疊的問題
- 修復備份大型檔案時卡頓的問題

## [1.7.0-beta1]

### 新增
- AppStore 全新 UI 改版，帶來更現代、清晰的應用程式瀏覽體驗
- 新增應用程式管理頁，集中管理已安裝應用程式、更新狀態與常用操作
- 新增應用程式探索頁，優化應用程式推薦、分類瀏覽與內容展示
- 新增原生 YAML 支援，提升設定編輯與解析體驗

### 優化
- 優化應用程式列表、分類頁面與應用程式詳情頁的展示體驗
- 優化容器日誌展示，提升日誌可讀性與問題排查效率
- 優化容器終端機操作，改進執行狀態展示與互動回饋
- 優化商用授權判斷邏輯，提升授權識別相容性與準確性
- 優化頁面快取清理策略，以提升頁面整體瀏覽體驗
