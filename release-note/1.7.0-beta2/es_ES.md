## [1.7.0-beta2]

### Fixed
- Se corrigió el problema por el que las listas de aplicaciones no se mostraban para los usuarios miembros
- Se corrigió el problema por el que las listas de aplicaciones no se cargaban en determinados entornos de red
- Se corrigió el problema por el que las aplicaciones no se actualizaban o instalaban debido a un manejo incorrecto de las comillas en las variables de entorno
- Se corrigieron los fallos de instalación de aplicaciones causados por un formato incorrecto de los valores de memoria en determinadas configuraciones regionales
- Se corrigió el problema por el que algunas ventanas no eran completamente visibles en dispositivos con pantallas pequeñas
- Se corrigió la superposición del recuento de archivos en la página de copias de seguridad
- Se corrigió la lentitud durante las copias de seguridad de archivos grandes

## [1.7.0-beta1]

### Added
- Se rediseñó la interfaz de AppStore para ofrecer una experiencia de exploración de aplicaciones más moderna y clara
- Se añadió una página de gestión de aplicaciones para administrar de forma centralizada las aplicaciones instaladas, el estado de las actualizaciones y las acciones comunes
- Se añadió una página de descubrimiento de aplicaciones con mejores recomendaciones, navegación por categorías y presentación de contenido
- Se añadió compatibilidad nativa con YAML para mejorar la edición y el análisis de configuraciones

### Improved
- Se mejoró la experiencia visual de las listas de aplicaciones, las páginas de categorías y las páginas de detalles de las aplicaciones
- Se mejoró la visualización de los registros de contenedores para ofrecer una mejor legibilidad y una resolución de problemas más eficiente
- Se mejoraron las operaciones del terminal del contenedor con un estado de ejecución más claro y una mejor retroalimentación interactiva
- Se mejoró la lógica de validación de licencias comerciales para reforzar la compatibilidad y la precisión del reconocimiento de licencias
- Se mejoró la estrategia de limpieza de la caché de páginas para mejorar la experiencia general de navegación
