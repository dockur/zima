## [1.7.0-beta2]

### Fixed
- S'ha corregit el problema pel qual les llistes d'aplicacions no es mostraven per als usuaris membres
- S'ha corregit el problema pel qual les llistes d'aplicacions no es carregaven en determinats entorns de xarxa
- S'ha corregit el problema pel qual les aplicacions no es podien actualitzar o instal·lar a causa d'un tractament incorrecte de les cometes a les variables d'entorn
- S'ha corregit el problema d'instal·lació d'aplicacions causat per un format incorrecte dels valors de memòria en alguns idiomes locals
- S'ha corregit el problema pel qual algunes finestres no es mostraven completament en dispositius amb pantalles petites
- S'ha corregit la superposició del recompte de fitxers a la pàgina de còpia de seguretat
- S'ha corregit la lentitud durant les còpies de seguretat de fitxers grans

## [1.7.0-beta1]

### Added
- S'ha redissenyat la interfície d'AppStore per oferir una experiència de navegació d'aplicacions més moderna i clara
- S'ha afegit una pàgina de gestió d'aplicacions per administrar de manera centralitzada les aplicacions instal·lades, l'estat de les actualitzacions i les accions habituals
- S'ha afegit una pàgina de descoberta d'aplicacions amb recomanacions millorades, navegació per categories i una presentació de contingut més cuidada
- S'ha afegit compatibilitat nativa amb YAML per millorar l'edició i l'anàlisi de configuracions

### Improved
- S'ha millorat l'experiència visual de les llistes d'aplicacions, les pàgines de categories i les pàgines de detall de les aplicacions
- S'ha millorat la visualització dels registres del contenidor per oferir una millor llegibilitat i una resolució de problemes més eficient
- S'han millorat les operacions del terminal del contenidor amb un estat d'execució més clar i una millor resposta interactiva
- S'ha millorat la lògica de validació de llicències comercials per augmentar la compatibilitat i la precisió del reconeixement de llicències
- S'ha millorat l'estratègia de neteja de la memòria cau de pàgines per millorar l'experiència general de navegació
