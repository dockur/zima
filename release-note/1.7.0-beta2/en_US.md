## [1.7.0-beta2]

### Fixed
- Fixed app lists not being displayed for member users
- Fixed app lists failing to load in certain network environments
- Fixed apps failing to update or install due to incorrect handling of quotation marks in environment variables
- Fixed app installation failures caused by incorrect memory value formatting in certain locales
- Fixed some windows not being fully visible on small-screen devices
- Fixed overlapping file counts on the backup page
- Fixed lag during large file backups

## [1.7.0-beta1]

### Added
- Redesigned the AppStore UI for a more modern and clearer app browsing experience
- Added an app management page to centrally manage installed apps, update status, and common actions
- Added an app discovery page with improved app recommendations, category browsing, and content presentation
- Added native YAML support to improve configuration editing and parsing

### Improved
- Improved the display experience for app lists, category pages, and app detail pages
- Improved container log display for better readability and more efficient troubleshooting
- Improved container terminal operations with clearer execution status and interaction feedback
- Improved commercial license validation logic to enhance license recognition compatibility and accuracy
- Improved page cache cleanup strategy to enhance the overall browsing experience
